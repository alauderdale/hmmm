transparent PNG support

The Problem

When building prototypes, it is common to incorporate things like a dropdown menu shared by multiple pages. Unfortunately, if we want to build out this interaction using inVision, we have to fake it with an entire screen to display the dropdown.

The Solution

By supporting transparent PNGs, we could simply load the PNG over the top of the screen from which the Transparent PNG was navigated from. This way, we can display something like a dropdown from the main nav across multiple pages.

Note: The transparent PNG screen would have its own set of hotspots!
